1. DESCRIPTION
-------------------------------------------------------------------------------
The tacklelib library additional cmake support modules to build a project from
Windows and Linux platforms separately but with the same configuration files and
variables.

The latest version is here: https://sf.net/p/tacklelib/cmake

WARNING:
  Use the SVN access to find out new functionality and bug fixes.
  See the REPOSITORIES section.
