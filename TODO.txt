scripts/tools/ToolAdaptors/vbs/update_shortcut_props_from_dir.bat,
scripts/tools/ToolAdaptors/vbs/reset_shortcut_from_dir.bat
-------------------------------------------------------------------------------

* Try to create a shortcut previous working directory path starting a root
  directory in a temporary directory and mount the path root directory to
  unexisted drive letter (using `mklink` or `junction.exe`, administrator
  permissions is required before the execution) from the previous working
  directory. If the drive letter already exist, then replicate the unexisted
  path using the existed drive letter to make the previous working directory
  exist.

  This will circumvent the read of a shortcut TargetPath property for unlinked
  drive or for a partially unexisted target path.
  The shell COM component interfaces prevent to read such paths and so to fix
  them.

  Add flags:
    -allow_replicate_unexisted_paths
