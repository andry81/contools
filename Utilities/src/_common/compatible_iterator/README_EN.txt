1. DESCRIPTION
-------------------------------------------------------------------------------
Compatible iterator implementation to support iteration over set of STL
compatible containers with different type. Input type of templated STL
compatible container can be different too.

-------------------------------------------------------------------------------
2. PREREQUISITES
-------------------------------------------------------------------------------

Currently used these set of OS platforms, compilers, interpreters, modules,
IDE's, applications and patches to run with or from:

1. C++11 compilers:

* (primary) Microsoft Visual C++ 2015 Update 3 or Microsoft Visual C++ 2017
* (secondary) GCC 4.7.1+
* (experimental) Clang 3.4.1+
